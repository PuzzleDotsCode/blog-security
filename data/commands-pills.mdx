* john the ripper: crack zip password (old versions)

```
# 1. get the hash
zip2john protected.zip > zip.hashes

# 2. john unhashing
john zip.hashes

$ ...
$ (21*paswd_21)      (protected.zip)
$ ...

# OPTIONAL: recover the password
john --show zip.hashes
$ protected.zip:21*passwd_21::protected.zip:files
```
