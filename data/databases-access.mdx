# Databases Access by CLI


* redis

```
redis-cli -h 10.129.111.33 -p 6379
```

* mysql: root

```
mysql --host=10.129.187.6 --user=root
```

* mongodb:

```
mongo 10.129.183.245:27017 
```

* psql

```
psql -h [host] -U [username] -p [port]

# commands
\list : show the list of databases
\c [database] : use the database
\d : list tables
\du+ : show users roles
\dt [table] : use a table

# commands should carry " ; " (semicolon)
SELECT * from flagtable;
```
